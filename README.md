# Trae网站应用程序

这是一个基于Flask和Microsoft SQL Server的Web应用程序，提供安全监控解决方案的企业网站。

## 技术栈

- **后端**：Python, Flask, SQLAlchemy
- **数据库**：Microsoft SQL Server
- **前端**：HTML5, CSS3, JavaScript
- **部署**：Gunicorn/Nginx, HTTPS支持

## 功能特性

- 企业展示网站
- 产品和服务展示
- 案例研究展示
- 联系和预约功能
- API接口支持
- 安全部署选项
